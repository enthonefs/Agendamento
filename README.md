# 📅 Sistema de Agendamento

Projeto desenvolvido para gerenciar agendamentos de serviços, permitindo o controle de clientes e horários disponíveis.

## 🚀 Funcionalidades

- Cadastro de clientes  
- Criação de agendamentos  
- Listagem de horários  
- Controle de disponibilidade  

## 🛠️ Tecnologias

- Java  
- Spring Boot  
- MySQL  
- Maven
- Docker

## ▶️ Como executar

```bash
git clone https://github.com/enthonefs/Agendamento.git
cd Agendamento
./mvnw spring-boot:run
